# 긴 계좌명 계좌 선택 레이아웃 수정

## 배경

기록 탭에서 거래를 등록할 때 계좌 선택 항목에 긴 계좌명이 표시되면 선택 토글의 오른쪽 아이콘이 옆으로 밀린다.

## 목표

- 선택된 계좌명이 길어도 계좌 선택 트리거의 오른쪽 아이콘 위치를 유지한다.
- 계좌명 텍스트만 한 줄 말줄임 처리한다.
- 브로커 로고와 placeholder 표시 동작은 유지한다.

## 구현

- `TradeBasicForm`의 계좌 선택 표시 영역에 `min-w-0`, `flex-1`, `overflow-hidden`, `truncate`를 적용한다.
- 브로커 로고는 줄어들지 않도록 고정하고 계좌명 텍스트만 줄어들게 한다.
- 드롭다운 항목도 같은 구조로 맞춰 긴 계좌명이 목록 레이아웃을 밀지 않게 한다.

## 검증

- 긴 계좌명을 선택해도 오른쪽 아이콘이 밀리지 않는지 확인한다.
- 브로커 로고가 있는 계좌, 없는 계좌, 미선택 placeholder가 정상 표시되는지 확인한다.
- `pnpm tsc --noEmit`로 타입 오류가 없는지 확인한다.
