## 배포 체크리스트 (출시 노트 외 운영 작업)

- **DB 마이그레이션: 해당 없음** — `api/alembic` 변경 없음 (app-v1.3.8_32..HEAD).
- **BE 배포: 기능상 불필요** — api 변경은 버전 bump(`pyproject.toml`)와 `.env.example` 문서뿐, 런타임/라우트/스키마 무변. ⚠️ 단 main push 시 Coolify 가 `api/` 변경 감지로 **동일 코드 재기동**할 수 있음(무해). API 계약 변경 없어 구버전 앱과 완전 하위호환.
- **MIN_SUPPORTED_VERSION: 변경 불필요** — 현재값 `1.3.0` 유지. 이번 릴리즈에 breaking API 변경 없음(네이티브 플러그인은 휴면, BE 계약 무변).
- **모바일 스토어 제출: 필요** — app 네이티브 변경(플러그인 3종 + 엔타이틀먼트 + google-services.json + AppDelegate) → 📱 재심사 필요. build 32→33.
- **실행 순서**: (마이그레이션·BE 배포 없음) → 스토어 제출.
